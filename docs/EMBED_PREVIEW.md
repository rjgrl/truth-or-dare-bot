# Embed preview (text mockups)

These are approximations of what players see in Discord.

## `/truth category:funny`

```
┌─────────────────────────────────────────────┐
│ 😂 Truth                                    │
│ **What is the dumbest thing you have done   │
│ to make someone laugh?**                    │
│                                             │
│ Category: Funny    Rating: PG13             │
│ Requested by: @Player                       │
│                                             │
│ [➡️ Next] [⏭️ Skip] [🎲 Random] [💬 Truth] [⚡ Dare] │
└─────────────────────────────────────────────┘
```

## `/party start`

```
┌─────────────────────────────────────────────┐
│ 🎉 Party Mode                               │
│ Round: 1                                    │
│ Current turn: @Player2                      │
│ Players: ▶️ @Player2 • @Player1 • @You      │
│ Completed: 0   Skipped: 0   Points: 0       │
└─────────────────────────────────────────────┘
```

Color accent uses `EMBED_COLOR` from `.env` (default Discord blurple `#5865F2`).
